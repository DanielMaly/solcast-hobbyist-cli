# SolHob: Solcast Hobbyist CLI

Work in progress.

## Usage

Clone this boilerplate by git:

```bash
git clone https://github.com/DanielMaly/solcast-hobbyist-cli.git -b main --depth 1
```

And installing dependencies by npm:

```bash
npm install
```

## CLI


```bash
npm create typescript-cli
```
